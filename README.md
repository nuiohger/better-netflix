# Better Netflix

## Build

##### Visual Studio Code
To build JavaScript from TypeScript:

CTRL + SHIFT + B

tsc: build


##### Build JavaScript for Browsers
Run build.sh

### Distribution
To create the zip files use the -z or --zip parameter.

### Testing
To build the extension without minification use the -d or --debug parameter.
